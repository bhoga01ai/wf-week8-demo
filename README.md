# wf-week08-demo

This project is a simple Python application that prints a "Hello" message.

## Project Structure

- `main.py`: The main entry point of the application.
- `requirements.txt`: A list of Python dependencies for this project.
- `pyproject.toml`: Project metadata and build system configuration.
- `.gitignore`: Specifies intentionally untracked files to ignore.
- `uv.lock`: A lock file for the `uv` package manager.
- `.python-version`: Specifies the Python version for the project.

## Getting Started

### Prerequisites

- Python 3.x
- `uv` package manager

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/bhoga01ai/wf-week8-demo.git
   ```
2. Navigate to the project directory:
   ```bash
   cd wf-week08-demo
   ```
3. Install the dependencies:
   ```bash
   uv pip install -r requirements.txt
   ```

### Usage

To run the application, execute the following command:

```bash
python main.py
```

This will print the following output to the console:

```
Hello from wf-week08-demo!
```
